# Visual Identity: Systems Craft

## Positioning

This is part personal site and part engineering portfolio. It should feel like a senior engineer's workspace: precise, calm, technically credible, and personal without becoming a formal resume.

## Principles

- **Clarity first:** layouts should make the engineering story easy to scan before asking visitors to read long prose.
- **Fast and static:** keep the main site simple, accessible, and durable; interactive complexity belongs inside project demos.

## Visual System

- **Color:** warm neutral page background, dark green-black text, muted green primary accent, copper secondary accent, and soft blue-gray technical metadata.
- **Typography:** system sans-serif stack with strong weight contrast, generous line height, and compact metadata labels.
- **Layout:** responsive max-width shell, wider than the old text column; dense but calm sections; no nested cards.
- **Surfaces:** use subtle borders and light panels for project entries, never heavy drop shadows or glossy gradients.
- **Imagery:** use the existing portrait as a deliberate homepage asset with clean cropping and concise supporting copy.
- **No eyebrows:** avoid standalone eyebrow, kicker, or overline labels above page and section headings.
